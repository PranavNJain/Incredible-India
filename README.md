## 📌 About

A fully responsive, single-file recreation of the Incredible India tourism website. Built as a frontend development project to demonstrate HTML5, CSS3, and vanilla JavaScript skills.

---

## ✨ Features

- Fully responsive layout (mobile + desktop)
- Fixed navigation with smooth scroll
- Hero section with gradient background
- Statistics counter section
- Destination cards with hover effects
- Heritage & culture section
- Cuisine showcase grid
- Trip planning form
- Indian tricolor theme throughout

---

## 🛠️ Tech Stack

| Technology | Usage |
|---|---|
| HTML5 | Semantic structure |
| CSS3 | Flexbox, Grid, animations, CSS variables |
| JavaScript (vanilla) | Scroll effects, nav highlight |

## 🚀 How to Run

No dependencies, no build step needed.

1. Clone or download the repo
2. Open `index.html` in any browser

```bash
git clone https://github.com/your-username/incredible-india.git
cd incredible-india
# Open index.html in browser
```
